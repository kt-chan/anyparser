OceanStor A800 is widely applicable to financial AI models, regional intelligent computing centers, AI computing power centers, scientific research platforms, autonomous driving platforms, and more.

# Product Features

# $\triangleleft$ Ultra Performance for Higher AI Cluster Computing Power Utilization

OceanStor A800 uses NPUDirect Storage (NDS) to build an optimal performance path from NPUs to data storage, bypassing complex protocol stacks and eliminating the need for data copy, improving performance by $40\%$ .

New-gen OceanStor A800 AI storage uses the DataTurbo high-performance file acceleration engine to directly read data from and write data to the storage system in kernel mode. Kernel-mode caching enables memory data mobility and halves memory usage. OceanStor A800 offers 500 GB/s bandwidth per controller enclosure, over 100 TB/s aggregated bandwidth per cluster, read and write for TB-level checkpoints in seconds when training models with trillions of parameters, and fast resumable training from checkpoints.

Based on a global unified data view, OceanStor A800 uses DPUs to offload the general computing power of CPUs. Data read and write operations are processed by DPUs, bypassing the CPUs and memory. The host can thus directly connect to disks to read and write data. A single controller enclosure provides 10 million IOPS and a cluster provides hundreds of millions of IOPS, with four times higher data loading efficiency than the leading alternative.

# $\triangleleft$ More Flexible Multi-Dimensional Expansion

Based on the brand-new hardware architecture, new-gen OceanStor A800 AI storage delivers flexible scalability with both scale-out and scale-up capabilities. A single cluster can scale out to up to 512 controllers and EB-level storage capacity, ensuring high performance and capacity density. In addition, computing cards, such as DPU and GPU accelerator cards*, can be inserted into the back end of the array. The storage cluster can scale up to a maximum of 4,096 cards, together to provide 14,400 TFLOPS of computing power and accelerate data processing. It can also carry multiple offloaded functions, including encryption, compression, and vector retrieval.

A single new-gen OceanStor A800 AI storage system can meet diverse ecosystem requirements. It is compatible with file, object, and vector, among other protocols, and supports the entire large AI model development process, such as data ingestion, preprocessing, model training, and deployment and inference. OceanStor A800 eliminates the need for data copy and accelerates the iteration and development of large AI models.

New-gen OceanStor A800 AI storage provides SmartQuota. With this file system quota technology, system administrators can manage and control the storage resources. By limiting the disk space that can be used by specified subjects, SmartQuota prevents excessive resource occupation and maximizes the value of storage space.

# $\triangleleft$ More Efficient Inference with AI-Native Data Paradigms

The data paradigms supported by new-gen OceanStor A800 AI storage range from NAS, and objects to new vectors and graph adaptive retrieval engines. OceanStor A800 enables high-performance retrieval from massive amounts of multimodal knowledge. The solution's competitive advantage lies in its inference based on RAG knowledge bases and Unified Cache. OceanStor A800 provides a built-in large-capacity AI knowledge base with hundreds of millions of data records for 3x higher QPS than the leading alternative. This enables faster knowledge base creation and more accurate queries.

OceanStor A800 significantly accelerates inference thanks to its Unified Cache feature, which offers multi-level KV cache from on-chip memory to SSDs. This establishes a PB-level global shared resource pool for KV cache, enabling flexible allocation across multiple compute nodes and xPUs and eliminating repeated computing. This slashes the Time to First Token (TTFT) by $78\%$ while improving the throughput by $60\%$ . It also uses sparse attention algorithms for retrieval to compress tokens to enable the inference of ultra-long sequences beyond the context window, ultimately making inference smoother, faster, and more cost-effective.
